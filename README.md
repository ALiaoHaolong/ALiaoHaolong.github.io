## 本地运行

克隆仓库

```shell
$ git clone https://github.com/ALiaoHaolong/ALiaoHaolong.github.io.git
```

进入目录

```shell
$ cd ALiaoHaolong.github.io
```

安装依赖（只能用 pnpm 安装依赖）

```shell
$ pnpm install
```

启动开发服务器

```shell
$ pnpm docs:dev
```
